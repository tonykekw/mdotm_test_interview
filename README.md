This is a Java Spring Boot Web Application made to manage a Pet Directory with a RESTful API and a web interface.

Features:

CRUD operations for pets via REST.

A simple web page made with Thymeleaf.

Spring Security for authentication.

Swagger for API documentation.

PostgreSQL/MongoDB as DB.

Basic tests made with Mockito.

Tech: Java 21, Spring Boot, Maven/Gradle, Thymeleaf, Spring Security, Swagger, PostgreSQL.

Clone: git clone https://github.com/tonykekw/mdotm_test_interview.git

Build: ./mvnw clean install
Run: ./mvnw spring-boot:run
Access UI: http://localhost:8080
Swagger: http://localhost:8080/swagger-ui.html
